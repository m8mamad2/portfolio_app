IoT LED Control Application using Flutter, ESP8266 and MQTT

This project is an IoT solution that allows you to control an LED connected to an ESP8266 board using a Flutter mobile application. The communication between the app and the ESP8266 is facilitated by MQTT (Message Queuing Telemetry Transport)
